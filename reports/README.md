# Reports

This directory stores reports generated by review agents and analysis tools.

## Purpose

Reports capture the output of automated code reviews, architecture audits, and other analysis. They provide a historical record of code quality and help track improvements over time.

## File Naming

Use date-prefixed names that indicate the report type:
- `2026-01-15-architecture-review.md`
- `2026-01-16-test-coverage-analysis.md`
- `2026-01-17-swift-safety-audit.md`
